# Tech Context

## Технологический стек
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Icons**: Lucide JS.
- **Data**: JSONL.
- **Dev Tools**: Python 3.x (для `run_app.py`).
- **CI/CD**: GitHub Actions, GitHub Pages.

## Технические ограничения
- На текущий момент проект является статическим сайтом.
- JSONL парсится целиком при загрузке, что может замедлять работу при экстремально больших объемах данных.

## Связь с общим стеком BHLTHY
- В глобальных правилах указан стек: Flutter, Dart Frog, MongoDB.
- ТЕКУЩИЙ ПРОЕКТ использует альтернативный стек. Требуется уточнение планов по миграции или интеграции.
