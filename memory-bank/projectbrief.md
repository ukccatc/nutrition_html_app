# Project Brief: Nutrition HTML App

Интерактивный исследователь данных о пищевой ценности продуктов (Nutritional Intelligence).

## Основные цели
- Визуализация данных из формата JSONL.
- Поиск и фильтрация продуктов.
- Предоставление детальной информации о витаминах, минералах и энергетической ценности.

## Текущее состояние
- Базовая версия на HTML/Vanilla CSS/JavaScript.
- Локальный сервер на Python для разработки.
- Настроен GitHub Actions для автоматического развертывания на GitHub Pages.
