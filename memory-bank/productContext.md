# Product Context

Проект предоставляет удобный интерфейс для изучения состава продуктов питания, помогая пользователям BHLTHY понимать пищевую ценность того, что они едят.

## Пользовательский опыт (UX)
- Современный, "стеклянный" (glassmorphism) дизайн.
- Плавные анимации при загрузке карточек.
- Мгновенная фильтрация по названию или группе продуктов.
- Интуитивно понятное отображение нутриентов с иерархией.

## Ключевые функции
- Прямой доступ к файлам данных (JSONL).
- Автоматическая загрузка `products.jsonl`.
- Детальный разбор разделов: Энергия, Витамины, Минералы.
