# System Patterns

## Архитектура
- **Client-Side Rendering (CSR)**: Весь UI строится динамически в браузере.
- **Data Format**: Использование JSONL (JSON Lines) для хранения данных о продуктах, что упрощает потоковую обработку больших наборов данных.
- **Server**: Легковесный Python HTTP-сервер для локального тестирования и обхода CORS ограничений.

## Структура файлов
- `index.html`: Единственная точка входа (SPA), содержит стили, разметку и логику.
- `products.jsonl`: База данных продуктов.
- `run_app.py`: Вспомогательный скрипт для локального запуска.
- `.github/workflows/deploy.yml`: Спецификация CI/CD.

## Дизайн
- Использование Google Fonts (Inter).
- Иконки Lucide.
- Адаптивная сетка (CSS Grid).
